accessibilityHazard

Thing > Property > accessibilityHazard

A characteristic of the described resource that is physiologically dangerous to some users. Related to WCAG 2.0 guideline 2.3 (WebSchemas wiki lists possible values).
  • Canonical URL: http://schema.org/accessibilityHazard
  • Leave public feedback on this term 💬
  • Check for open issues.
Values expected to be one of these types
Text
Used on these types
CreativeWork
